# Code of Conduct

## Our standards

- Be respectful and constructive.
- Focus criticism on code and decisions, not people.
- Avoid harassment, discrimination, and personal attacks.
- Assume good intent and ask clarifying questions when needed.

## Enforcement

Project maintainers may remove comments, close threads, or block accounts for violations. Repeated violations can result in permanent exclusion from the project spaces.

## Reporting

Report conduct issues privately to `support@CRAVAB.com`.
